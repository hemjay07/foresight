### Farming Prevention

**Problem:** Once users know about airdrop, they'll spam predictions (bots, low-quality entries).

3. **Anti-Bot Metrics:**
   - Limit 10 predictions per day per user
   - Require CAPTCHA at signup
   - Flag suspicious patterns (50 predictions/day)
   - Require email verification (not just wallet)

4. **Airdrop Decay:**
   - Predictions made after airdrop announcement: 0.5x weight
   - Only predictions from Month 1-3 count (founding period)
   - Heavy users who start in Month 4 get less allocation

5. **Manual Review of Top 100:**
   - Check top airdrop recipients for suspicious patterns
   - If caught farming, remove from airdrop (nuclear)

## 6. NETWORK EFFECTS & FLYWHEEL DESIGN

### The Broken Flywheel (Why Most Products Fail)

Most apps assume:
```
More Users → Positive Network Effects → Even More Users
```

**But this only works if network effects are real.**

In prediction markets, network effects are **weak**:
- More users don't make my predictions better
- More users don't make markets more liquid
- More users don't give me more information

So the flywheel breaks at ~500-1000 DAU (the LTV/CAC wall).

### The Foresight Flywheel (Status-Based, Not Network-Based)

**Our network effect is psychological, not technical.**

```
┌─────────────────────────────────────────┐
│ USER JOINS                              │
│                                         │
│ → Makes 3 predictions (low friction)    │
│                                         │
│ → Sees leaderboard (their rank: #10K)   │
│                                         │
│ → Gets first CS: 25                     │
│                                         │
│ → PSYCHOLOGICAL LOOP:                   │
│                                         │
│ "I need to beat #9,999. Just one more   │
│  prediction. Just one more win."        │
│                                         │
│ → Predicts daily (habit)                │
│                                         │
│ → Climbs to #5,000 (visible progress)   │
│                                         │
│ → Shares on Twitter ("62% win rate!")   │
│                                         │
│ → Friends see → Some join               │
│                                         │
│ → More users on leaderboard             │
│                                         │
│ → Original user climbs higher           │
│ (because there's more room at bottom)   │
│                                         │
│ → More competitive → More daily use     │
│                                         │
│ → MORE USERS (organic referral)         │
│                                         │
└─────────────────────────────────────────┘
```

### The Exact Network Effects (Not Vibes, Real Math)

**Effect 1: Leaderboard Liquidity**
- More users = deeper leaderboard
- When you're at #5K with 500 DAU, you're "top 1%"
- At 5K DAU, you're only "top 20%"
- This makes climbing harder, so you predict more

**Effect 2: Market Depth**
- More users = more betting volume per market
- More volume = better odds (tighter spreads)
- Better odds = you're more likely to win
- Win = you stay = your friends join

**Effect 3: Information Density**
- More users = better aggregate prediction
- "Wisdom of crowds" means markets more accurate
- More accurate settlement = people trust the platform
- Trust = daily habit

**Effect 4: Influencer Engagement**
- More users predicting = more attention to influencers
- Influencers notice (Foresight mentions their names)
- Some engage ("I should try this")
- Now influencers are invested = more organic promotion

### Where the Flywheel Breaks (The Crisis Points)

#### Break Point 1: 0 → 50 DAU (The Void)
**Problem:** No leaderboard, no status, no reason to come back.

**Solution:**
- Seed with 20-30 influencers Day 1
- They post their CS, create artificial activity
- First 50 real users see a "living" leaderboard
- Breaks through the void

#### Break Point 2: 50 → 500 DAU (The Retention Wall)
**Problem:** Leaderboard is deep (everyone's #2K+). Climbing feels pointless. No reason to predict tomorrow.

**Solution:**
- Weekly reset (give everyone a chance to climb)
- Streak mechanics (7-day, 30-day bonuses)
- Revenue share offers (month 3-4: "If you hit CS 150, you earn $100/month")
- Monetization hint ("Your CS = your airdrop allocation")

#### Break Point 3: 500 → 2K DAU (The Liquidity Squeeze)
**Problem:** Market depth is thin. A big bet moves odds too much. Early predictors get bad odds.

**Solution:**
- Introduce AMM (automated market maker) to smooth odds
- Increase # of markets per day (5 → 10)
- Add market types (head-to-head, ranges, not just binary)
- Premium tier: higher bet limits (create tier of competitive users)

### Critical Metric to Watch: D1 Retention

```
Target: 10% D1 retention (1 in 10 users comes back day 2)
Industry: 3-5% is normal for games
For us: Prediction accuracy is random, so 10% is realistic

If D1 < 5%:
- Product problem (not habit-forming)
- Messaging problem (wrong audience)
- UX problem (too complex)

If D1 > 10%:
- We're onto something
- Scale immediately
```

## 9. RETENTION MECHANICS: THE DAILY HABIT LOOP

### The Challenge: Prediction Markets Are Passive

Unlike social apps (Netflix, TikTok), prediction markets require:
1. **Active decision-making** ("What do I think will happen?")
2. **Real stakes** (money, even if small)
3. **Waiting for settlement** (can't know if you won immediately)

This is high friction. Most users won't check daily.

**Solution:** Engineer daily habits through psychological mechanisms.

### Mechanism 1: The Streak (Status Anxiety)

```
"7-day streak 🔥"

Your streak is ACTIVE. Come back tomorrow to keep it going.

If you miss tomorrow: RESET TO 0

Visual design:
┌─────────────────────┐
│ 🔥 STREAK: 7 DAYS   │
│                     │
│ ⏰ Next market      │
│    opens in: 6h     │
│                     │
│ Don't break it      │
│ [Make prediction]   │
└─────────────────────┘

Psychological hook:
- Progress (7 > 6 > 5 > 0) is visible
- Loss aversion ("I don't want to break it")
- Tomorrow's action is pre-committed
```

**Data proves it works:**
- Duolingo: Streaks drive 500% higher retention
- Snap: Snap streaks (people snap daily) are their #1 retention feature
- Our forecast: 7-day streak → 40% daily return rate, 14-day → 60%

### Mechanism 2: The Leaderboard (Social Pressure)

```
Leaderboard position: #347

Your rank changed: ↓ 12 positions

Now you're: #359 globally

In your friend group:
┌─────────────────────┐
│ #1  @friend_a (CS89)│
│ #2  YOUR NAME (CS78)│
│ #3  @friend_b (CS67)│
└─────────────────────┘

"Make one more prediction to beat @friend_a"
```

**Psychological hook:**
- Social comparison (who's beating me?)
- Visible progress (rank is a number)
- Competitiveness ("just one more prediction")

### Mechanism 3: The Win (Dopamine Hit)

```
Market settled ✅

You predicted YES on @ethereum
Engagement: 487,342 (target was 500K)

RESULT: LOSS (-$5)

Your win rate: 62% (above average!)
```

**Wait, we're showing a loss?**

Yes, but:
- Win rate is above 50% (positive feedback)
- Encourage "one more prediction to boost win rate"
- Show daily progress ("You were 58% yesterday, now 62%!")

### Mechanism 4: The Monetization Signal (The Real Hook)

**Most powerful:**

```
Your Credibility Score: 247 (Silver)

At this rate, you'll earn $50-100/month
when we launch revenue share (Month 4).

Your monthly prediction earnings:
- 62% win rate × $5 avg bet × 3.5 predictions/day
- × 30 days = $325/month potential

Keep your streak. Keep winning.
```

**Psychological hook:**
- Money signal (quantified upside)
- Future orientation ("In 3 months...")
- Path to wealth ("If I keep this up...")

### Mechanism 5: The Daily Market Opening (Trigger)

Push notification (6 AM UTC):

```
🎯 Today's Markets Are Open

3 new prediction markets just dropped.
Your leaderboard position: #347

Come make your predictions.
Don't break your 7-day streak.

[Open App]
```

**Timing matters:**
- 6 AM is morning (coffee + phone)
- Early = you see market before it trends
- First-mover advantage (early predictors get better odds)

**Frequency:**
- Once per day (6 AM)
- No more (avoid notification fatigue)

### Mechanism 6: The Weekly Challenge (Sustained Engagement)

Every Saturday:

```
🏆 Weekly Challenge: Top 100

This week: Predict influencer engagement in real-time.

Your weekly standing: #347 → goal is top 100

You have 2 days left.

To reach top 100, you need:

- That's roughly 8-10 wins in a row

Can you do it?

[See Weekly Leaderboard]
```

**Psychological hook:**
- Time-limited (weekend push)
- Achievable goal (top 100, not #1)
- Competitive frame ("Can you do it?")

Share your achievement:

## 11. THE "WHAT IF WE'RE WRONG" PLAN: PIVOT OPTIONS

### Scenario 1: Prediction Markets Don't Gain Traction (<50 DAU M2)

**Signal:** Users sign up, make 1-2 predictions, disappear. No streak. No leaderboard climbing.

**Root cause:** Prediction accuracy is random. Users don't feel like they're winning. No skill component.

**Pivot Option A: Betting Exchange**

Instead of fixed-odds markets, become a **peer-to-peer betting exchange** (like Polymarket but for CT metrics).

```
Users create their own markets:
- @alice bets YES on @vitalik engagement
- @bob takes the other side (NO)
- No house = no rake conflict
- We just match bets + take 1% fee

Why this is better:
- Users feel like they're trading (vs gambling)
- More liquidity (users make markets)
- Less regulatory (decentralized model)
- Better UX (see other people's odds)
```

**Why it works:** Uniswap killed Kyber with this model (decentralized > centralized).

**Implementation:** 2-3 weeks

**Expected impact:** 3-5x increase in DAU (betting feels different than gambling)

#### Pivot Option B: Influencer Governance DAO

Instead of prediction markets, focus on **credibility scoring as an investment thesis**.

```
Foresight DAO:
- Community votes on "who's credible"
- Voting = stake (you earn if you're right)
- Top voters earn governance tokens
- Platform manages $100K+ in influence derivatives

Why it works:
- Less random (crowd wisdom > individual luck)
- More community (voting > betting)
- Clearer monetization (token sales, derivatives trading)
```

**Implementation:** 1 month

**Expected impact:** 50+ power users, $500K TVL within 3 months

### Scenario 2: CT Influencers Don't Care (<20 seed influencers)

**Signal:** Outreach to 50 influencers. Only 2-3 respond. No organic adoption.

**Root cause:** Influencers see us as parasitic (betting on them without permission). Bad PR risk.

**Pivot Option A: Influencer Collaboration Platform**

Position Foresight as **Influencer Collab Hub** instead of prediction market.

```
Features:
- Influencers post their collaboration requests ("Looking for DeFi guest post")
- Other creators respond + get visibility
- Foresight takes 10% of partnership fees

Why influencers care:
- Direct monetization (vs just being predicted on)
- Audience access (other creators see their requests)
- Professional network (like LinkedIn for creators)

Why it works:
- Flip the incentive (help creators, don't predict on them)
- Clearer monetization (partnership fees)
- Creator economy is hot
```

**Implementation:** 4-6 weeks

**Expected impact:** 500+ influencers on platform, $20K GMV within 3 months

#### Pivot Option B: B2B Analytics for Brands

Position Foresight as **Influencer Analytics Tool** for brands.

```
Brands want to know:
- Which influencers are most credible?
- Whose engagement is fake?
- Who's on the rise?

Foresight sells:
- Influencer credibility reports ($500/month)
- Audience quality scores
- Engagement trend analysis

Why it works:
- Brands have budgets (vs individual users)
- Influencer fraud is a real problem
- Clear ROI story ("Save money on fake influencers")
```

**Implementation:** 2-3 weeks

**Expected impact:** 20+ brand customers, $10K MRR within 6 months

**Signal:** Cease-and-desist letter from SEC/FTC (unlikely but possible)
