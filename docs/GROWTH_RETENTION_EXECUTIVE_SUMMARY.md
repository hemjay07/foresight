## THE CORE PROBLEM (2 minutes to understand)

**The Gap:**
- Weekly contests mean users draft once, then... nothing until final day
- 5-6 days of silence = users forget the app exists
- No reminders, no updates, no social pressure
- Result: 80%+ churn at day 5-7

**Why This Matters:**
- User enters team Monday 12:00 UTC
- Next action available: Sunday 23:59 UTC (or they check randomly)
- No dopamine hits during the week
- Friends don't create pressure (no leaderboard comparison)
- Referral system exists but completely invisible

**Current State vs. Desired:**
```
CURRENT: Draft ✓ → [Dead Zone: 5 days] → Score updates (sparse) → Contest ends
DESIRED: Draft ✓ → Daily check-ins (4 score updates + friend activity) → Competition → Celebrate → Re-draft
```

---

## THE SOLUTION (3 mechanisms)

### Mechanism 1: Score Updates (4x Daily)
**What:** Real-time feedback on team performance
- Mon/Tue/Wed/Thu/Fri/Sat/Sun: Scoring happens at 12:00, 18:00, 00:00, 06:00 UTC
- User gets email/push: "You gained +42 pts! You're #14 now."
- Score updates are 4 reasons to check the app (instead of 0)

**Psychology:** Variable reward schedule + immediate feedback (Robinhood effect)

**Frequency:** 4 interruptions/week interrupt the dead zone

### Mechanism 2: Social Leverage (Follow + Friends Tab)
**What:** Make friends visible competition
- Follow button on leaderboard (cyan highlight)
- "Friends" tab on Compete page shows only people you follow
- See exact gap: "@sarah beat you by 47 pts"

**Psychology:** Loss aversion + local status competition (stronger than global rank)

**Data:** DraftKings shows 2-3x engagement boost from friend leaderboard

### Mechanism 3: Progression Visibility (XP + Levels)
**What:** Show users they're making progress, even if not winning
- Home page: "You're Level 5, 85/100 XP to Level 6"
- Leaderboard: "Lvl 12" badge next to each player
- Profile: Full quest list (completed ✓ and upcoming)
- Every action has XP reward (enter contest +50, draft team +25, etc.)

**Psychology:** Gamification + mastery progression (Duolingo effect)

**Engagement:** Users check progress even when not placing well

## THE ENGAGEMENT LOOP (90 seconds, then weekly cycle)

### Onboarding: 90 Seconds to Value
```
30s: Email/social login (Privy handles this)
20s: Create username + avatar
30s: See real leaderboard with live data
20s: Optional follow of #1 player
30s: Enter free contest + draft 5-player team
  └─ [Share on Twitter] button (viral moment #1)
30s: See live score updating (real-time feedback)

Total: 90 seconds from signup to "my team is earning points right now"
```

This is your hackathon demo narrative.

### Weekly Cycle: 7 Phases

| Phase | Duration | User Action | Emotional Driver | Checks/Week |
|-------|----------|-------------|------------------|-----------|
| **Draft** | 12h (Mon 12-24) | Pick team + share | Excitement + FOMO | 1 |
| **Locked** | 24h (Tue 00) | See initial score | Hope/Anxiety | 2-3 |
| **Scoring Live** | 72h (Tue-Thu) | Watch team climb | Momentum | 5-7 |
| **Jockeying** | 96h (Thu-Sun) | Check rank vs friends | Competitive tension | 3-4 |
| **Finale** | 24h (Sun) | Final score incoming | FOMO/Urgency | 8-10 |
| **Finalized** | 24h (Mon 00+) | Claim prize, share | Closure + aspiration | 2 |
| **Re-entry** | Ongoing | Draft next week | Fresh momentum | 1+ |

**Expected check-ins:** 12+ per week (vs. 1-2 without strategy)

## 5 RE-ENGAGEMENT TRIGGERS (The Tactical Layer)

These 5 moments pull users back during the week:

### Trigger #1: Score Updates (4x/day)
- **When:** 12:00, 18:00, 00:00, 06:00 UTC (every 6h average)
- **Message:** "You gained +42 pts! You're #14 now."
- **Frequency:** 4 per day × 7 days = 28 notifications/week
- **Why it works:** Variable rewards (don't know if +12 or +120)

### Trigger #2: Friend Activity (Real-time)
- **When:** ActivityFeedCard updates every 30 seconds
- **Message:** "@alex_tr +289 pts | You +42 pts | @newbie +87 pts"
- **Why it works:** Social proof (FOMO seeing others winning)

### Trigger #3: Competitive Position (Wed-Fri)
- **When:** If rank changes significantly
- **Message:** "@sarah just passed you! You're #15. Close gap: 47 more pts to #14."
- **Why it works:** Loss aversion + achievable goal (-47 pts is concrete)

### Trigger #4: Finale Countdown (Sun 23:00 onwards)
- **When:** Final 24 hours, then hourly Sun evening
- **Message:** "ONE DAY LEFT! You're #14. Final scores lock soon."
- **Why it works:** Temporal scarcity (24h > 1 week in psychological urgency)

### Trigger #5: Prize Claim + Celebrate (Mon 00:00+)
- **When:** Immediately after contest finalization
- **Message:** "YOU PLACED #14! Claim prize. [Share Victory]"
- **Why it works:** Endowment effect (SOL in wallet) + social proof (sharing)

## 6 VIRAL MOMENTS (The Growth Layer)

These moments make users text friends about the app:

### Viral Moment #1: Draft Celebration + Share Card (PRIMARY)
```
User finishes draft → Sees formation visual
→ [Share on Twitter] button (gold, prominent)
→ Pre-filled tweet mentions @captain_name + team + prize pool
→ Tweet shows: "Just drafted @alex_tr as captain in CT Fantasy! Competing for 0.05 SOL"
→ @alex_tr (100K followers) might retweet → exponential reach
```

**Network effect:** Influencer co-sign = viral growth

### Viral Moment #2: Victory Share
```
User places top-3 → Sees medal icon + SOL prize
→ [Share Victory] button
→ Tweet: "Just won on @ForesightGame! #2 placement, earned 0.002 SOL 🏆"
→ Friends see: "Wait, real money for just picking influencers?"
```

**Conversion:** 12-15% of viewers sign up

### Viral Moment #3: Captain Boost Realization
```
Monday 18:00 UTC: @captain_name tweets
→ User's score jumps +117 pts (1.5x multiplier visible)
→ User goes from #47 to #15 in one update
→ TEXT: "Bro I picked @alex_tr as captain and he just tweeted = +117 pts = +32 spots"
```

**Leverage moment:** Makes captain role feel powerful

### Viral Moment #4: Friend Challenge
```
User sees friend @bestie_ct in #10 spot
→ Clicks → Sees their winning team composition
→ [Challenge This Team] button
→ Auto-generates share: "@bestie_ct just beat me by 150 pts. I'm challenging them next week."
→ Creates weekly micro-competition
```

**Local competition:** Drives week 2, 3, 4 signups (not just week 1)

### Viral Moments #5 + #6: Influencer Mentions + Formation Visual

**#5 — Influencer Mention:** If user's drafted team performs well, other fans see it

**#6 — Formation Visual:** Users show friends because "this UI is actually beautiful"

## QUESTS + XP: PROGRESSION VISIBILITY (No 5th Nav Item!)

**The Problem:** Progress page exists but is completely hidden. Users have no idea they're leveling up.

**The Solution:** Distribute progression across existing pages:

| Page | What Shows | Update Frequency |
|------|-----------|-----------------|
| **Home** | XP bar + recent quests | Every 30s |
| **Profile** | Level badge + full quest list | Every 30s |
| **Leaderboard** | "Lvl 12 ⭐" next to each name | Every 60s |
| **Draft** | Active quests (pick captain, etc.) | Real-time |
| **Contest Detail** | Potential XP rewards | Static |

**Result:** Users see they're making progress everywhere they look (without adding a 5th nav item)
