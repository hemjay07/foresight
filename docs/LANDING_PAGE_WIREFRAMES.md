## ANIMATION SEQUENCES

### On Scroll-Into-View (Formation Card)
```
Timeline: 0ms → 300ms
────────────────────

0ms:   opacity: 0, transform: translateY(32px)
       ↓ (CSS transition: 300ms ease-out)
300ms: opacity: 1, transform: translateY(0)

Curves:
  ease-out = faster at start, slower at end
  (feels snappy, like product is "greeting" you)
```

### Score Card (100ms stagger)
```
Timeline: 100ms → 400ms
──────────────────────

100ms: opacity: 0, transform: translateY(32px)
       ↓ (CSS transition: 300ms ease-out)
400ms: opacity: 1, transform: translateY(0)

Follows formation by 100ms
```

### Leaderboard Card (200ms stagger)
```
Timeline: 200ms → 500ms
──────────────────────

200ms: opacity: 0, transform: translateY(32px)
       ↓ (CSS transition: 300ms ease-out)
500ms: opacity: 1, transform: translateY(0)

Follows score by 100ms
Creates cascade effect
```

### Score Bars Fill (staggered)
```
Timeline: Formation appears at 0ms
Score bars fill as card appears

Bar 1 (Activity):    0ms delay   → 400ms animation
Bar 2 (Engagement):  80ms delay  → 400ms animation
Bar 3 (Growth):      160ms delay → 400ms animation
Bar 4 (Viral):       240ms delay → 400ms animation

Easing: ease-out (snappy)
Total visible animation time: 640ms
(feels alive without being slow)
```

## RESPONSIVE BREAKPOINTS

### Desktop (≥1024px)
- Asymmetric bento grid (60/40 split)
- Formation card: 380px height
- Score/Leaderboard cards: 180px each
- Padding: 80px top/bottom sections
- Gap between cards: 20px

### Tablet (768px to 1023px)
- Grid collapses to single column
- Formation card: 320px height (scaled down)
- Cards full width
- Padding: 64px top/bottom sections
- Gap between cards: 16px
- No visible change at 768px; smooth transition

### Mobile (375px to 767px)
- Single column stack
- Formation card: 280px height (mobile optimized)
- Cards full width minus padding
- Padding: 32px top/bottom sections
- Gap between cards: 16px
- Tier grid: 4 columns (tight but works at 375px)
