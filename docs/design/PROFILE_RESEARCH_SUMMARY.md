# Profile Page Research Summary — What Judges See

> **Quick visual guide to competitive profile design insights**
> **One read = understand the entire redesign strategy**

---

## THE CORE INSIGHT

### How DraftKings/FanDuel Users See Their Profile

```
First Glance (< 1 second):
├─ "What's my score?" → Look at big gold number
├─ "What's my rank?" → Look at color/badge
└─ "Am I doing well?" → Compare to leaderboard

Visceral Reaction:
├─ ❌ "Just some stats" = forgets it exists
├─ ✅ "That's a REAL score!" = feels proud, shares it
```

### How Crypto Users See Their Profile

```
First Glance:
├─ "Is this person verified?" → Look for checkmark
├─ "Are they serious?" → Look for credentials (Founder, badges)
├─ "Should I follow/trust them?" → Check social proof (followers)

Visceral Reaction:
├─ ❌ "No proof of legitimacy" = doesn't trust
├─ ✅ "On-chain verified, founder status" = credible, follows
```

### How Judges See Your Profile (What We Need)

```
First Glance (< 1 second):
├─ Big gold number (score)
├─ Rank + credentials (founder, verified)
├─ Clean layout (professional polish)

Verdict:
├─ ❌ Current: "Okay there's a profile" (neutral)
├─ ✅ Target: "Wow this person is #8 and legit" (impressed)
```

## THE PSYCHOLOGY BEHIND THE REDESIGN

### Status Signaling (Why It Matters)

```
BEFORE (Current):
Profile page shows you have:
├─ Some stats
├─ Some teams
├─ Some history
└─ Feeling: "Okay, I guess I'm a player"

AFTER (Redesigned):
Profile page shows you have:
├─ #8 All-Time Rank ← Proof of skill (top 1%)
├─ Founder Status ← Proof of access (exclusive)
├─ Verified Badge ← Proof of legitimacy (on-chain)
├─ Win Rate 40% ← Proof of consistency
└─ Feeling: "WOW, I'm a SERIOUS player!"
```

**The Psychological Shift:**
- Before: Profile is informational (neutral)
- After: Profile is aspirational (impressive)

---

### Identity Theory (Why Metrics Matter)

In competitive games, **the metric becomes your identity**.

```
Current Foresight:
├─ You are: "A player with 1,135 FS"
└─ Feeling: Okay, that's the number?

Redesigned Foresight:
├─ You are: "The #8 Ranked Player"
├─ Proof: Founder #18, On-Chain Verified
└─ Feeling: I'm LEGIT and others know it
```

**Evidence:**
- DraftKings: "I'm an 8,950 point player" (identity from score)
- FanDuel: "I'm MVP status" (identity from tier)
- Reddit: "I'm 10K karma" (identity from metric)

**What This Means for Judges:**
When a judge sees "#8 All-Time Rank" prominently displayed, their brain goes: "This person has proven skill. Legit winner."

---

### Social Proof Theory (Why Follower Counts Matter)

```
BEFORE:
├─ Profile shows: 0 followers, 3 following
└─ Judge thinks: "Is anyone even following this player?"

AFTER (with follow button):
├─ Profile shows: 18 followers, 3 following ← Real users watching
└─ Judge thinks: "Other players trust this person. Must be good."
```

**The Tipping Point:**
- 0-5 followers: Feels lonely
- 5-20 followers: Credible
- 20+ followers: Authority

You'll hit this through the follow button social features (Phase 2).

## INFORMATION ARCHITECTURE: The Hierarchy

### What Judges See in First 1 Second

```
CURRENT:
1. "Your Profile" heading
2. ... avatar placeholder
3. ... hmm, let me scroll

TARGET:
1. Gold number: 1,135
2. Cyan rank: #8
3. Badges: Founder, Verified
→ Judge thinks: "Impressive, next."
```

### The Redesigned Hierarchy

```
LAYER 1 (Highest Priority — Hero):
├─ Foresight Score: 1,135 pts (gold, huge)
└─ Rank: #8 All-Time (cyan, large)

LAYER 2 (High Priority — Credibility):
├─ Founder #18 (badge, gold)
└─ ✓ Verified on Tapestry (badge, cyan)

LAYER 3 (Medium Priority — Context):
├─ Win Rate: 40% (2W, 3L)
├─ Streak: 4 weeks
└─ Followers: 2, Following: 3

LAYER 4 (Lower Priority — Details):
├─ Tabs: Overview, Teams, History, Watchlist
├─ This Week: +245 pts
├─ Level: NOVICE (0 XP)
└─ [Everything else scrollable below]
```

**The Rule:** Each layer is 1/2 the visual importance of the layer above.

## QUICK MENTAL MODEL

### The Sorare Card Model (What We're Adopting)

Sorare made NFT cards feel valuable by:
1. **Clear rarity visual** (colored frame, scarcity numbers)
2. **Ownership proof** (contract address visible)
3. **Market value** (price shown)
4. **Clean layout** (everything fits on one card)

### Foresight Profile (Adapted)

We're making profiles feel valuable by:
1. **Clear rank visual** (gold score, cyan rank)
2. **Credentials proof** (badges: Founder, Tapestry)
3. **Social proof** (followers, following)
4. **Clean layout** (hero section, then tabs)

**Same Psychology, Different Domain.**
