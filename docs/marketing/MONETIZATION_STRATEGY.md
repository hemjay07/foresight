---

### **Phase 2: Freemium** (Months 2-4)
**Goal**: $10K-$50K MRR
**Revenue Source**: Paid tournaments, premium features
**Still Free**: Basic gameplay, referral rewards

**What We Launch**:

1. **Paid Tournaments** (Like DraftKings)
   ```
   Entry: $5-$20
   Prize pool: 90% of entries
   Platform take: 10%

   Example: 100 players × $10 = $1,000 pool
   - Winner: $450 (45%)
   - Top 10: $450 (45%)
   - Platform: $100 (10%)
   ```

   **Referral Hook**: Give inviter 5% rakeback on friend's entries
   - Friend enters $20 tournament → You get $1 credit
   - Incentivizes inviting players who'll pay

2. **Premium Badges** (Cosmetic)
   ```
   Cost: $2.99 one-time
   - Exclusive designs
   - Animated badges
   - Custom colors

   Referral Hook: 10% off if invited friend buys
   ```

3. **Private League Hosting** (Creator Economy)
   ```
   Hosts charge: $10/person entry
   Platform takes: 20% ($2)
   Host keeps: 80% ($8)

   Example: Host creates 50-person league
   - Total: $500
   - Platform: $100
   - Host: $400

   Referral Hook: Free hosting for users with 50+ active invites
   ```

**Expected Revenue**:
- 10K users × 10% conversion = 1K paying
- Avg $10/month = $10K MRR
- Growing to $50K as tournaments scale

### **Phase 3: Token Launch** (Months 4-6)
**Goal**: Reward early users, create liquidity
**Revenue Source**: Token launch, treasury management
**Model**: Blur airdrop playbook

**What We Launch**:

1. **$FORESIGHT Token** (On Base)
   ```
   Total Supply: 1,000,000,000 (1B tokens)

   Distribution:
   - 40%: Community (airdrops, rewards)
   - 20%: Team (4-year vest)
   - 15%: Investors (future rounds)
   - 15%: Treasury (liquidity, growth)
   - 10%: Public sale

   Initial Market Cap: $5M (if $0.005/token)
   ```

2. **Airdrop Calculation** (Rewarding Early Users)
   ```
   Airdrop Pool: 400M tokens (40%)

   Criteria:
   - Founding Members (First 1K): 5x multiplier
   - XP earned: 1 token per 100 XP
   - Referral quality score: Multiplier (1.0x to 3.0x)
   - Active weeks: 1.2x per week played
   - Tournament wins: Bonus allocation

   Example User:
   - Founding Member #234: 5x multiplier
   - 50,000 XP earned: 500 base tokens
   - Quality score 80%: 2.0x multiplier
   - 8 active weeks: 1.96x multiplier
   - Total: 500 × 5 × 2.0 × 1.96 = 9,800 tokens

   If token launches at $0.01: User gets $98 airdrop
   If it goes to $0.10: $980 value
   ```

3. **Why This Works**:
   ✅ Early users rewarded (founding members get most)
   ✅ Engagement rewarded (XP converts to tokens)
   ✅ Quality rewarded (spam invites get nothing)
   ✅ Active users rewarded (playing > just signing up)
   ✅ Creates liquidity event (users can sell or hold)

**Messaging During Phase 1-2**:
- "Founding members will be rewarded" ← Vague but true
- "Quality score matters for future rewards" ← Airdrop hint
- "Building your position" ← Accumulating XP
- "Early supporters get more" ← First 1K get 5x

**Revenue Impact**:
- Token sale (10% public): $5M at $0.005
- Treasury appreciates as platform grows
- Creates exit liquidity for early team/angels

### **Phase 4: Play-to-Earn** (Months 6-12)
**Goal**: Sustainable P2E model (NOT ponzi)
**Revenue Source**: Tournament fees, token utility, brand deals
**Model**: Sustainable FPL + Crypto rewards

**What We Launch**:

1. **Token Utility**
   ```
   $FORESIGHT uses:
   - Tournament entries (10% discount vs USD)
   - Premium features (exclusive to holders)
   - Governance (vote on rule changes)
   - Staking (earn yield from tournament fees)
   ```

2. **Earnings Model** (Sustainable)
   ```
   How Players Earn:
   - Win tournaments: Prize pools (90% of entries)
   - Complete quests: Token rewards from treasury
   - Rank top 100: Monthly rewards
   - Refer active users: 5% of their tournament fees

   How We Make Money:
   - 10% tournament rake
   - Brand sponsorships (influencer deals)
   - Premium subscriptions
   - NFT sales (special achievements)
   ```

3. **Why NOT a Ponzi**:
   ✅ Revenue from tournaments (not new users)
   ✅ Sponsorships (real money in)
   ✅ Platform take is sustainable (10% rake)
   ✅ Token backed by treasury + revenue
   ✅ Users earn from SKILL (tournament wins)
   ✅ NO multi-level (referrals are 1 level deep)

**Expected Revenue**:
- 10K users × $20 avg tournament spend/month = $200K
- Platform rake (10%): $20K/month
- Sponsorships: $10K/month
- Premium subs: $5K/month
- Total: $35K/month = $420K/year

```

→ Fear: I'll miss the airdrop if I don't engage NOW

```
